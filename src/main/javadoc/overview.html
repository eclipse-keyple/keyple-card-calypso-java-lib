This document is the specification of the API dedicated to the integration of the Calypso Card Extension in aKeyple Application.

Overview

Use {@link org.eclipse.keyple.calypso.CalypsoCardExtensionProvider#getService()} to get the unique instance of the {@link org.eclipse.keyple.calypso.CalypsoCardExtension} interface containing all the entry points provided by the extension:

  • Creation of a selection case targeting a specific PO to be provided to the Keyple Selection Service.
    {@link org.eclipse.keyple.calypso.CalypsoCardExtension#createPoSelection(PoCardSelector poCardSelector)}
    A selection case targets a specific PO defined by the POJO {@link org.eclipse.keyple.calypso.po.PoCardSelector} and possibly supplemented with commands to be executed after the selection (see methods of {@link org.eclipse.keyple.calypso.po.PoCardSelection}).
  • Creation of specific SAM Card Resource profiles with the Keyple Card Resource Service.
    {@link org.eclipse.keyple.calypso.CalypsoCardExtension#createSamCardResourceProfileExtension createSamCardResourceProfileExtension ()} to be referenced in the PO security settings {@link org.eclipse.keyple.calypso.CalypsoCardExtension#getPoSecuritySettingBuilder(java.lang.String samProfileName)}.
  • Transaction service to perform non-secure operations with a PO.
    {@link org.eclipse.keyple.calypso.CalypsoCardExtension#createPoUnsecuredTransaction(org.eclipse.keyple.core.service.Reader reader, PoSmartCard poSmartCard)}
  • Transaction service to perform secure operations with a PO (Calypso Secure Session).
    {@link org.eclipse.keyple.calypso.CalypsoCardExtension#createPoSecuredTransaction(org.eclipse.keyple.core.service.Reader reader, PoSmartCard poSmartCard, PoSecuritySetting poSecuritySetting)}

The PO data collected at each step (selection, transaction) are stored in dedicated POJOs whose interfaces are available in the {@link org.eclipse.keyple.calypso.po} package.
