This document is the specification of the API dedicated to the integration of the Calypso Card Extension in a Keyple Application.

All the entry points of the extension are provided by the {@link org.eclipse.keyple.calypso.CalypsoCardExtension} interface.

{@link org.eclipse.keyple.calypso.CalypsoCardExtensionProvider#getService()} is to be used to get the unique instance of the extension.

The entry points are:

  • One method to create a selection case to be provided to the Keyple Selection Service. {@link org.eclipse.keyple.calypso.CalypsoCardExtension#createPoSelection(CalypsoPoCardSelector calypsoPoCardSelector)}
    It targets a specific PO defined by the POJO {@link org.eclipse.keyple.calypso.po.CalypsoPoCardSelector} and possibly supplemented with commands to be executed after the selection (see methods of {@link org.eclipse.keyple.calypso.po.CalypsoPoCardSelection}).
  • Two methods to configure the SAM resource service needed to process secure operations with POs.
    {@link org.eclipse.keyple.calypso.CalypsoCardExtension#getSamResourceService()} and {@link org.eclipse.keyple.calypso.CalypsoCardExtension#getPoSecuritySettingBuilder(java.lang.String samProfileName)}.
  • Two methods to provide two types of transaction services: non-secure and secure.
    {@link org.eclipse.keyple.calypso.CalypsoCardExtension#createPoUnsecuredTransaction(org.eclipse.keyple.core.service.Reader reader, CalypsoPoSmartCard calypsoPoSmartCard)}
    {@link org.eclipse.keyple.calypso.CalypsoCardExtension#createPoSecuredTransaction(org.eclipse.keyple.core.service.Reader reader, CalypsoPoSmartCard calypsoPoSmartCard, PoSecuritySetting poSecuritySetting)}

The PO data collected at each step (selection, transaction) are stored in dedicated POJOs whose interfaces are available in the {@link org.eclipse.keyple.calypso.po} package.
